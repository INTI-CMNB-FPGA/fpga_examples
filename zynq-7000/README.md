# Zynq-7000 examples

This directory contains examples/tutorials which can be easily re-targeted to support any Zynq-7000 based board.

## Abbreviations

* PL: Programmable Logic (FPGA)
* PS: Processor System (Processor)
* DMA: Direct Memory Access
